Detailed Description

This class represents a genealogical graph that inherits from the /ref graph::Graph class and additionally assigns all the vertices within the graph to their levels.

The level of a vertex is defined recursively as follows: 1) All the proband individuals are assigned to level 0. 2) If the maximal level among a vertex's children is n, then the vertex's level is n + 1. In other words, the level of a vertex is the length of the longest path from a proband to this vertex in a graph. Notice that this definition assumes that the probands cannot be ancestor one to another! (TODO: Consider checking this scenario)
